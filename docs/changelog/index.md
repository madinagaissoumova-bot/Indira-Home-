# Changelog - Indira Home

Ce changelog enregistre les travaux livres par zone de travail.

Une entree doit rester factuelle : elle decrit un changement deja realise et verifie. Le changelog ne sert pas a planifier des fonctionnalites futures.

## Convention

Chaque fichier de zone utilise ce format :

```md
## AAAA-MM-JJ - Titre du travail livre

- Changement concret livre.
- Verification ou impact notable.
```

Quand un travail touche plusieurs zones, ajouter une entree courte dans chaque fichier concerne.
