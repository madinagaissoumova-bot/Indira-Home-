# 2026-06-02 - Direction editoriale premium

## Objectif

Donner a Indira Home une direction plus editoriale, minimaliste et premium, inspiree des codes mode haut de gamme sans copier une marque existante.

## Intention visuelle

- Palette plus blanc casse, noir profond et neutres chauds.
- Moins d'effets roses visibles, moins d'ombres lourdes.
- Header et navigation plus fins, plus mode.
- Cartes produit plus sobres, centrees sur la photo.
- Admin plus net, plus utilitaire et plus premium.

## Portee

- CSS global.
- Parcours client public.
- Espace admin.

## Hors portee

- Changement de logique metier.
- Changement de donnees.
- Refonte des composants React sauf necessite.

## Verification

- `npm run lint`
- `npm run build`
- Verification responsive des pages principales.
