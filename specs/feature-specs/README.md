# Feature Specs

Ce dossier contiendra une fiche detaillee par fonctionnalite.

Chaque fiche devra decrire :

- Objectif de la fonctionnalite
- Donnees necessaires
- Actions possibles
- Boutons
- Messages
- Erreurs
- Cas speciaux
- Regles metier

Les fiches sont redigees progressivement, une fonctionnalite a la fois.
