# Global Spec - Indira Home

## Description simple

Créer un site internet en russe pour **Indira Home**, une boutique qui vend principalement des articles pour la maison : vaisselle, electromenager, decoration et objets du quotidien.

Le catalogue doit pouvoir s'elargir plus tard a d'autres categories de produits si la boutique evolue.

Le site doit presenter les produits de la boutique et permettre aux clientes de passer une commande simplement.

## Qui utilise le site ?

### Clientes

Le site est utilise principalement par des femmes de tout age vivant en Republique tchetchene, ou achetant pour une personne situee en Republique tchetchene.

Elles utiliseront principalement le site depuis un telephone mobile. Elles doivent pouvoir comprendre rapidement quels produits sont disponibles et faire une commande sans processus complique.

### Admin / vendeuse

Le site est aussi utilise par la personne qui gere la boutique.

Elle doit pouvoir presenter ses produits, recevoir les commandes et organiser la confirmation avec les clientes.

## Pourquoi ce site existe ?

La vendeuse a besoin d'un endroit clair et professionnel pour montrer tous ses produits.

Les clientes ont besoin d'un moyen simple pour voir les articles disponibles, lire les descriptions des produits, connaitre les prix et passer commande sans devoir demander toutes les informations manuellement.

## Quel probleme le site resout ?

Aujourd'hui, les produits peuvent etre difficiles a presenter, a retrouver et a commander de facon organisee.

Le site resout ce probleme en centralisant les produits dans un catalogue accessible, en russe, avec un parcours de commande simple.

## Ou ca se passe ?

- Pays : Russie
- Region principale : Republique tchetchene
- Zone de livraison initiale : Republique tchetchene uniquement
- Langue du site : russe

## Perimetre global

La premiere version doit rester simple : montrer les produits disponibles et permettre a une cliente d'envoyer une commande sans creer de compte et sans paiement en ligne.

Elle concerne une boutique unique, avec une offre principalement orientee maison : vaisselle, electromenager, decoration et objets du quotidien.

Le site doit rester assez souple pour permettre l'ajout d'autres categories plus tard.

Le projet n'a pas vocation a devenir une marketplace avec plusieurs vendeurs. Il est pense pour une seule boutique : Indira Home.

Le paiement en ligne n'est pas necessaire dans la premiere version. Il pourra etre ajoute plus tard si le projet evolue.

La V1 doit aussi inclure un espace admin complet pour gerer les produits, les categories, le stock et les commandes.

## Nom de la boutique

- Indira Home
